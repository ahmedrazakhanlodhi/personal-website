Projects

MortgageMatch

MortgageMatch is a consumer-facing Python command-line tool that produces a clean HTML report to analyze U.S. mortgage approval vs. denial patterns in Illinois (2024) using public Home Mortgage Disclosure Act (HMDA) data. It compares outcomes across borrower profiles (income and loan-amount bands, loan purpose), counties, and lenders to surface where outcomes diverge and where additional review is warranted.
